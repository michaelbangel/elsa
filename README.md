# MJ23R2 Financial Risk Analytics Framework

## Overview

MJ23R2 is an R-based financial analytics framework designed to evaluate:

- Variance between reported and verified financial data
- Settlement outcomes and distortion risk
- Operational and negotiator behavior patterns
- Litigation exposure indicators
- Fraud risk indicators (risk-based, not deterministic)
- Financial Integrity Index (FII)
- Post-settlement re-exposure risk

## Purpose

This project demonstrates end-to-end analytical thinking:
data → risk signals → governance decisioning.
## Key Output

- Financial Integrity Index (0–100)
- Fraud Risk Indicator (0–100)
- Governance Priority Level
- Settlement Distortion Indicator
